# Toast

일시적으로 표시되는 간결한 메시지이다.

<Canvas of={ToastStories.UsageExample} />

사용 예시 코드는 다음과 같다.

```tsx
import { Button } from '@/components/commons/Button';
import useToast from '@/hooks/useToast';

const { show } = useToast();

<Button onClick={() => show('산책 기록이 저장되었습니다.')}>저장하기</Button>;
```

`useToast`가 반환하는 객체에 있는 것은 토스트를 표시하는 `show` 함수 하나이다. `show`의 매개변수는 다음과 같으며 `text`만 필수이다.

<Markdown>
    {`
| 매개변수  | 타입   | 설명                        |
| --------- | ------ | --------------------------- |
| text      | string | 토스트에 표시될 텍스트      |
| duration? | number | 토스트가 표시되는 시간 (ms) |
`}
</Markdown>
